# Social News API

A RESTful API backend for a social news platform, built with Node.js. This API provides core functionality for user management, article publishing, commenting, and topic-based organization.

## Features

- User authentication and authorization
- Create and manage articles
- Comment on articles
- Organize content by topics
- User profiles
